# 2.  HTML初体验

```html preview
<!DOCTYPE html>
<html lang="zh-CN">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>HTML初体验</title>
</head>
<body>
  <!-- 页面标题 -->
  <h1>欢迎来到 HTML 的世界！</h1>

  <!-- 段落文本 -->
  <p>HTML 是构建网页的基础语言，学会它，你可以轻松打造自己的网页。</p>

  <!-- 图片 -->
  <img src="https://www.w3.org/html/logo/downloads/HTML5_Badge_512.png" 
       alt="W3C HTML5 Logo" 
       width="50" >

  <!-- 超链接 -->
  <p>点击这里访问 <a href="https://developer.mozilla.org/zh-CN/docs/Web/HTML" target="_blank">MDN HTML 教程</a> 学习更多 HTML 知识。</p>

  <!-- 有序列表 -->
  <h2>HTML 的基本结构包括：</h2>
  <ol>
    <li><strong>DOCTYPE 声明</strong>：告诉浏览器使用 HTML 的版本。</li>
    <li><strong>html 标签</strong>：网页的根元素。</li>
    <li><strong>head 标签</strong>：定义网页的头部信息。</li>
    <li><strong>body 标签</strong>：定义网页的主要内容。</li>
  </ol>

  <!-- 无序列表 -->
  <h2>学习 HTML 的好处：</h2>
  <ul>
    <li>轻松创建个人博客或作品展示网站。</li>
    <li>为学习前端开发打下坚实的基础。</li>
    <li>提升对互联网的理解。</li>
  </ul>

  <!-- 按钮 -->
  <button onclick="alert('你已经迈出了学习 HTML 的第一步，继续加油！')">点我加油</button>
</body>
</html>
```
